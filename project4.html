MM Degree

M Degree tells the story of two pastry chefs that met working together and decided to start they own company as a team. My roll was to create a desing that focus users' attention on their distinct work

www.mmdegree.com

UI Design

The interface design main concept is to focus users' attention on the distinct work. The goal is reached applying minimal design conceps, by making the images the main elements, by using a sober color pallete.

UX/UI: User Flow, Style Guide, Interface Design, Interaction Design

Web Development

The design is implemented using React.js library that alows better performance for its rendering by using a virtual DOM. Having the code structured in components also gives me the flexibility to combine CSS Libraries without overwriting each other, by using the components from React Materialize and React Bootstrap

Frontend Development: React.js, HTML, CSS, JavaScript, JS6, Bootstrap 4, Materialize

Responsive Design
